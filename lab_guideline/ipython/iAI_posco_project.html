3. 필요 항목 기입

  • 필요 항목 기입 예시 (Box를 클릭 후, 해당 내용 기입)



  • 항목 기입 시

    • 연구목표, 연구내용 등은 Kick-off 미팅 자료에 내용을 토대로 기입, 혹은 Counter Part에게 문의해서 기입하면 됨.

  • 가장 까다로운 부분은 연구비 편성

    • POSCO System의 연구비 계산 방식과, POSTECH의 연구비 계산 방식이 다름. 따라서 연구비 차이가 발생함.

    • 이로 인해, 다음과 같은 주의사항이 있음.

  • 연구비 편성 주의사항
    1) 기존에 합의된 연구비 총액은 연구계획서의 연구비 내역서에서만 같으면 됨.
    $\quad$(ex. 기존에 합의된 연구비 총액 1억 5천만원, 연구계획서의 연구비 내역서 총액 1억 5천만원.)
    2) 기존에 합의된 연구비 총액과 POSCO System의 연구비 총액은 달라도 괜찮음.
    $\quad$(다만, 기존에 합의된 연구비를 넘어서면 안 됨.)
    $\quad$(ex. 기존에 합의된 연구비 총액 1억 5천만원, POSCO System의 연구비 총액 1억 4천 9백 2십 만원.)
    3) POSCO System 상의 인건비는 참여연구원 등록에서 M/M (프로젝트 참여기간) 을 조정해서 변경 가능.



4. 간접비/인건비 특성

1) 연구책임자 (교수님) 의 인건비는 100% 간접비로 흡수

2) 제경비는 100% 간접비로 흡수
$\;\;\;$총 인건비 (교수님 + 학생) 의 89%로 계산. 단, 비율은 매년 변동되므로 이혜령 선생님에게 학인이 필요함.

3) 기술개발비의 50%는 간접비로 흡수
$\;\;\;$총 인건비 제경비 5%로 계산

4) 과제 종료 후 잔액
$\;\;\;$인건비 잔액의 80%만 적립 가능 (인건비 잔액 20% 간접비로 흡수)
$\;\;\;$직접비 및 기술개발비 잔액은 100% 간접비로 흡수
